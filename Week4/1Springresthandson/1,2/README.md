
# Spring Learn Project

## ✅ Steps to Run in VS Code

1. Open VS Code.
2. File → Open Folder → Select this `spring-learn` folder.
3. Run Maven Package:
```bash
mvn clean package
```
4. Run Application:
```bash
mvn spring-boot:run
```
5. Output will show:
```
Application Starting...
(Application logs...)
Application Started Successfully.
```

## ✅ Dependencies Used:
- Spring Boot DevTools
- Spring Web
